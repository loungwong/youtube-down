# youtube-down
下载YouTube视频
2024年8月29日更新。

原理是利用在线视频转换网站来解析YouTube(油管)视频地址，从而实现下载YouTube(油管)视频的目的。有的网站可以下载1080p及以上清晰度视频，比如2k、4k、8k，前提是视频本身上传时具有该清晰度。下载的视频含声音。高清晰视频播放器推荐PotPlayer。
